# 🌈 战战的学习乐园

### ▶ 在线玩（点开即玩，手机/平板/电脑都行）：**https://tinymq.github.io/julienk12learning/**

一组给孩子（K1–K2 学龄前 / 小学低年级）用的**纯网页小游戏**，覆盖 **语文 / 数学 / 英语** 三个科目。
全部是单个 HTML 文件，**无需安装、无需下载、打开就玩**，用浏览器打开即可。

> 用 Claude Code 陪孩子做的家庭学习项目，公开分享。欢迎自取、克隆、改造给自己家娃用 😊

---

## 🎮 怎么玩

- **最简单**：下载整个文件夹，双击 [`index.html`](index.html)，从目录页点卡片进入任意游戏。
- **在线玩（推荐）**：直接访问 **https://tinymq.github.io/julienk12learning/** ，任何设备打开就能玩，把网址发给谁都行。
- 单独玩某个游戏：直接双击对应的 `科目-名称.html` 文件。

建议用 **电脑 + 键盘**，很多游戏需要敲键盘；平板也能玩点击类的。

## ➕ 如何新增一个游戏

保持体系整洁，遵循两步：

**1. 命名规则** —— 文件名统一为 `科目-游戏名.html`
- 科目前缀只用三种：`语文-` / `数学-` / `英语-`（以后要加科目在此列出，如 `科学-`）
- 例：`数学-加减法闯关.html`、`英语-单词消消乐.html`

**2. 登记到目录页** —— 打开 [`index.html`](index.html)，在对应科目的 `<div class="cards"> … </div>` 里，复制一张卡片改三处即可：

```html
<a class="card" href="科目-游戏名.html">
  <span class="emoji">🎲</span>          <!-- 换个 emoji -->
  <div class="title">游戏名</div>          <!-- 改标题 -->
  <div class="desc">一句话说明玩法。</div>   <!-- 改描述 -->
  <span class="go">开始 ▶</span>
</a>
```

别忘了把该科目标题里的「N 个游戏」计数 +1。

**3. 提交**
```bash
git add .
git commit -m "新增游戏：xxx"
git push
```

## 🌐 开启在线玩（GitHub Pages，可选）

1. 打开仓库 **Settings → Pages**
2. Source 选 `Deploy from a branch`，Branch 选 `main` / 根目录 `/ (root)`，保存
3. 等一两分钟，访问 `https://tinymq.github.io/julienk12learning/` 即可在线玩，把这个网址发给谁都能直接玩
